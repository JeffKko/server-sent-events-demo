# server sent events demo

This is a demo for Web Api => ```server sent events```

It's a chat room build on ```Vite```

see [demo page](https://server-sent-event-demo.herokuapp.com/) on heroku

## Installation

```bash
npm install
```

## Usage

dev on local

```bash
npm run dev
npm run dev:server
```

for production env

```bash
npm run build
npm start
```
